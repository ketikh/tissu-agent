# Tissu Shop — მფლობელის სახელმძღვანელო

ეს გზამკვლევი ხსნის როგორ ვმართოთ ბოტი და მაღაზია ყოველდღიურად.

## საწყისი გვერდი (Dashboard)

`/admin` მისამართზე გადასვლა გადახვალ „მთავარ" გვერდზე, სადაც ერთი შეხედვით ხედავ:

- **დღეს · შეკვეთები** — რამდენი ახალი შეკვეთა შემოვიდა დღეს
- **დღეს · შემოსავალი** — დღევანდელი ჯამური ფასი
- **სულ · შემოსავალი** — ყველა დროის
- **აქტიური მარაგში** — რამდენი მოდელია ხელმისაწვდომი
- **ბოლო 7 დღე** — გრაფიკი ყოველდღიური შეკვეთების და შემოსავლის
- **⚠️ დაბალი მარაგი** — მოდელები, სადაც 1 ან 0 ცალია. აქვე შეკვეთო მუდმივად.
- **ბოლო აქტივობა** — ბოლო 8 შეკვეთა სტატუსით

## მარაგის მართვა

### ახალი პროდუქტი
1. სარიდბარში აირჩიე შესაბამისი კატეგორია (ჩანთები, ყელსაბამები, ...)
2. ზემო მარჯვნივ „+ ახალი მოდელი" / „+ ახალი ყელსაბამი"
3. ჩანთებისთვის ავტომატურად ითხოვს ფხრიწიანი/თასმიანი და პატარა/დიდი
4. სხვა კატეგორიებისთვის ცარიელი card-ი გაჩნდება — ფასი შიგნით ჩაწერე

### ფოტოების ატვირთვა
- card-ის photo slot-ზე დააწკაპე — ფაილის ასარჩევი გაიხსნება
- ან drag-and-drop — ფოტო პირდაპირ slot-ზე გადაიტანე
- „+ სხვა რაკურსი" tile — მრავალ რაკურსს უჭერ მხარს
- Hover-ზე × ღილაკი გამოჩნდება — ფოტოს წაშლა
- ცარიელ slot-ზე × — slot-ი მთლიანად იმალება (მხოლოდ იმ პროდუქტზე)

### ფასდაკლება
1. card-ზე დააჭირე 🏷️ „ფასდაკლებაში" ღილაკს
2. „ახალი ფასი" input-ში ჩაწერე ფასდაკლებული ფასი
3. ✓ ღილაკი დააჭირე ან Enter
4. „ფასდაკლება" tab-ში ჩანს ყველა აქციური პროდუქტი
5. კლიენტი თუ „ფასდაკლება გაქვთ?" დაწერს, ბოტი ავტომატურად ამ ტაბიდან გადააცილებს

## შეკვეთების მართვა

„შეკვეთები" ტაბში ჩანს ყველა აქტიური შეკვეთა. სტატუსები:

| სტატუსი | რას ნიშნავს |
|---------|-------------|
| ⏳ Pending | ახალი შეკვეთა. ჯერ არ ჩამორთმეული. |
| 📦 Ready | შეფუთული, კურიერისთვის მზადაა. **მარაგი ავტომატურად მცირდება** ამ სტატუსზე გადასვლისას. |
| 🚚 Sent | კურიერი წავიდა |
| ✅ Delivered | მიწოდებული. ჩამოდის აქტიურიდან. |

**სტატუსის შესაცვლელად** — drop-down-ზე დააწკაპე.

**შეკვეთის წაშლა** — ✕ ღილაკი მწკრივის ბოლოს. დაადასტურე.

**ძებნა** — შეკვეთების ზემოთ input-ი. სახელი, ტელეფონი, კოდი — ფილტრავს ცხრილს პირდაპირ.

## ჩატები

„💬 ჩატები" ტაბში ბოლო 200 საუბარი. თითოეული row-ი ღილაკია:
- არხი (FB / IG / WA)
- conversation id
- ბოლო კლიენტის მესიჯი
- დრო

დააწკაპე → მარჯვნივ drawer იხსნება სრული ჩატის ისტორიით (clean-ად, ყველა system-tag-ის გარეშე).

## ინსაიტები

„📊 ინსაიტები" — სამი ცხრილი:

1. **⚠️ ხარვეზები** — კლიენტის უკმაყოფილო მესიჯები („არასწორი", „ოპერატორი", „არ მიყვარს"). row-ზე დააწკაპე → სრული ჩატის ისტორია.
2. **❓ ხშირი კითხვები** — განმეორებადი კითხვები რაოდენობით. კანდიდატი ბოტის პრომპტში ჩასაწერ პასუხებად.
3. **🛍️ მოთხოვნილი პროდუქტები** — რას გვეკითხებიან რაც არ გვაქვს (ipad ქეისი, კოსმეტიკის ჩანთა და ა.შ.). მომავლის მინიშნება რა დავამატო.

**გამოიყენე კვირაში ერთხელ** — ხარვეზებს ნახე, ბოტის რეაგირება გააუმჯობესო.

## კატეგორიები

### ახალი კატეგორიის შექმნა
1. სარიდბარში „+ ახალი კატეგორია" ღილაკი (Catalog სექციის ბოლოს)
2. შეავსე სახელი (მაგ: „ქამრები"), emoji (🔗), slug („belts")
3. ველების სია — თუ გინდა კონკრეტული ველები ქონდეს ამ კატეგორიის პროდუქტებს (ფერი, მასალა, სიგანე)
4. „შენახვა" → sidebar-ში გამოჩნდება

### არსებული კატეგორიის ველების რედაქტირება
- კატეგორიის pill-ზე hover — პატარა ⚙️ ღილაკი გამოჩნდება
- დააჭირე → რედაქტირების modal — სახელი, emoji, ველები, ან წაშლა

## WhatsApp შეტყობინებები

ბოტი გიგზავნის WhatsApp-ზე როცა:
- 🛒 ახალი შეკვეთა შემოვა (კოდით, ფასით, ტელეფონით, მისამართით + ადმინ პანელის ლინკით)
- 📷 კლიენტი ფოტოს გააგზავნა (ბოტი ვერ ცნობს) → „confirm" / „deny" ღილაკები ბმულად
- 🔗 ლინკი გააგზავნა
- 📋 გადახდის სქრინი → confirm/deny ბმულებით
- ⚠️ ტექნიკური შეფერხება

**დაუდასტურე/უარვყავი ფოტო** — WhatsApp-ზე confirm/deny ბმულზე დაწკაპუნებით. ბოტი ავტომატურად უპასუხებს კლიენტს.

## დღიური Routine

1. **დილით** — Dashboard: ახალი შეკვეთები?
2. **ყოველ შეკვეთაზე** — სტატუსი განაახლე (Pending → Ready → Sent → Delivered)
3. **WhatsApp შეტყობინებები** — უპასუხე/დაადასტურე სანამ საბრუნი ხდება
4. **კვირაში ერთხელ** — ინსაიტები: ხარვეზი ნახე, FAQ-ზე პასუხი გასცი
5. **თვეში ერთხელ** — პროდუქცია/ფასდაკლება გადახედე
